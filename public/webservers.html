Web Servers

What is a Web Server?

In basic terms, a (software) web server is an app that accepts requests over the web and serves data or files as a response.

Usually accessed via a browser over HTTP requests. The web server will manipulate the request in some way and then send a response back to the browser.

A Web Server:

  • Accepts requests (sent)
  • Servers a response (get)
  • Acessed with a HTTP requests via the Browser
